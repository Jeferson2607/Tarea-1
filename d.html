Default


Configuración predeterminada de un programa o sistema cuando no se especifica otra opción.


DNS


(Domain Name System) Sistema que traduce nombres de dominio en direcciones IP.


Diodo


Componente electrónico que permite el paso de corriente en una sola dirección.


Digitalizar


Proceso de convertir información analógica en formato digital.


Directorio


Estructura utilizada para organizar archivos en un sistema de almacenamiento.


Download


Descarga de datos o archivos desde un servidor remoto a una computadora local.


DVD


Disco óptico utilizado para almacenar datos, videos y software.
