Back door


(Puerta trasera) Método no documentado de acceso a un sistema informático, que puede ser explotado por hackers o administradores.

Ir a un video


Backup


UCopia de seguridad de datos importantes para prevenir la pérdida de información en caso de fallos o ataques informáticos.


Banner


Imagen o gráfico publicitario que aparece en páginas web con el objetivo de atraer la atención del usuario.


Base de datos


Colección organizada de datos que pueden ser accedidos, gestionados y actualizados por un sistema de gestión de bases de datos (DBMS).


Binario


Sistema numérico de base 2 que utiliza solo los dígitos 0 y 1, fundamental en la computación digital.


BIOS


(Basic Input/Output System) Firmware que inicia el hardware de una computadora y permite cargar el sistema operativo.


Bit


Unidad mínima de información digital, que puede representar un 0 o un 1.

Byte


Conjunto de 8 bits que representa un solo carácter de información.


Browser


(Navegador web) Software que permite acceder y visualizar contenido en Internet, como Google Chrome o Mozilla Firefox.


Boot


Proceso de arranque de un sistema operativo cuando se enciende una computadora.


Buffer


Área de memoria temporal utilizada para almacenar datos en espera de ser procesados.

Bug


Error o fallo en un programa informático que provoca comportamientos inesperados.
