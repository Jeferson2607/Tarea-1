Chat


Servicio de comunicación en línea que permite la conversación en tiempo real mediante texto.


Cracker


Individuo que utiliza sus conocimientos informáticos para vulnerar sistemas de seguridad con intenciones maliciosas.

Chip


Circuito electrónico miniaturizado integrado en dispositivos electrónicos como computadoras y teléfonos.


Compilador


Programa que traduce el código fuente de un lenguaje de programación a lenguaje máquina ejecutable por la computadora.


Comprimir


Reducción del tamaño de un archivo o conjunto de datos mediante algoritmos de compresión.

Cookie


Pequeño archivo de texto que los sitios web almacenan en la computadora del usuario para guardar información sobre su actividad en el sitio.


CPU


(Unidad Central de Procesamiento) Componente principal de una computadora que ejecuta instrucciones de programas.


Cursor


Indicador visual en la pantalla que muestra la posición actual en un documento o interfaz gráfica.
